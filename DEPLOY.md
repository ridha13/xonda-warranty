# كيفية رفع المشروع على Vercel

## الخطوات السريعة:

1. احذف الـ Repository القديم من GitHub
2. أنشئ Repository جديد
3. ارفع **جميع** الملفات من هذا المجلد مباشرة (بدون مجلدات إضافية)
4. اربط مع Vercel
5. لا تغير أي إعدادات
6. اضغط Deploy

## أو:

### الطريقة الأسهل:
استخدم Vercel CLI:

```bash
# ثبت Vercel CLI
npm i -g vercel

# في مجلد المشروع
cd xonda-frontend-only

# رفع مباشر
vercel
```

سيطلب منك تسجيل الدخول، ثم سيرفع المشروع تلقائياً!
